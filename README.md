# Student Performance Analysis using NumPy

## Description
This project analyzes student marks data using NumPy. 
It demonstrates core NumPy operations like slicing, axis-based computations, sorting, and conditional logic.

## Concepts Used
- NumPy arrays
- Axis operations
- Statistical functions
- Sorting & ranking
- Conditional grading

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the Jupyter notebook

## Output
- Subject-wise statistics
- Student rankings
- Grade assignment
